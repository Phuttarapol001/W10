#W1 
Shopping Cart
สร้าง แอปที่มีฟีเจอร์ กรอกข้อมูลสินค้า แสดงรายการสินค้าในตะกร้า
สามารถลบสินค้าที่ไม่ต้องการออกจากตะกร้าได้
แสดงยอดรวมราคาสินค้า

CartItem ใช้กำหนดรูปแบบสินค้าแต่ละชิ้นในตะกร้า

CartState ใช้กำหนดรุปแบบ state ของตะกร้า

initial State เป็นค่าเริ่มต้นตะกร้า

Reducers จะมี addItem, removeItem, clearCart

#W2 
To-Do List

สร้างแอป To-Do List สามารถเพิ่มงาน และลบงานที่ไม่ต้องการออก


Todo ใช้กำหนดรุปแบบข้อมูลของงาน

TodoState ใช้กำหนด stateสำหรับเก็บรายการ todo

initial State เป็นค่าเริ่มต้น

Reducers จะมี action addTodo, toggleTodo, removeTodo
